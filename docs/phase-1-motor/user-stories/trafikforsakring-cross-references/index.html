Trafikforsakring — Cross-Reference Index

Trafikforsakring (mandatory third-party liability) requirements are spread across multiple epics. This page provides a consolidated index linking to all trafikforsakring-related content, so that compliance reviewers and developers can find everything from one place.

For the dedicated trafikforsakring user stories, see Trafikforsakring User Stories. For the consolidated use case, see UC-TRF-001: Trafikforsakring Compliance Lifecycle.

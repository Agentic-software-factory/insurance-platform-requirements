6. Lessons Learned​

IDTitleKey Insight
LESSON-001Stale git index causes local build failuresRun git reset --hard origin/main at session start to avoid stale index state from parallel worktrees

Additional operational insights:

  • GitHub API 502 errors: The API can return 502 while still executing the merge. Always check PR state after receiving a server error before retrying.
  • Worktree branch conflicts: --delete-branch on gh pr merge can fail when other worktrees reference the branch. Use gh pr merge --squash without --delete-branch and clean up manually.

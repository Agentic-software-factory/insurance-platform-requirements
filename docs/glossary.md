# Glossary

Key terms used throughout the TryggFörsäkring requirements documentation.

| Term                   | Definition                                                                      |
| ---------------------- | ------------------------------------------------------------------------------- |
| **Trafikförsäkring**   | Mandatory third-party liability insurance (Trafikskadelagen 1975:1410)          |
| **Halvförsäkring**     | Partial comprehensive insurance (tier 2)                                        |
| **Helförsäkring**      | Full comprehensive insurance (tier 3)                                           |
| **Bonusklass**         | No-claims discount class                                                        |
| **Huvudförfallodag**   | Policy anniversary / renewal date                                               |
| **Ångerrätt**          | 14-day cooling-off right                                                        |
| **FSA**                | Finansinspektionen — Swedish Financial Supervisory Authority                    |
| **GDPR**               | General Data Protection Regulation                                              |
| **IDD**                | Insurance Distribution Directive                                                |
| **Transportstyrelsen** | Swedish Transport Agency (vehicle registry, insurance verification)             |
| **TFF**                | Trafikförsäkringsföreningen — mandatory membership for trafikförsäkring sellers |
| **BankID**             | Swedish digital signing standard                                                |

# TryggFörsäkring — Insurance Platform Requirements

Business requirements documentation for TryggFörsäkring AB, built with [Docusaurus 3](https://docusaurus.io/).

## Getting Started

### Prerequisites

- Node.js >= 20.0

### Installation

```bash
npm install
```

### Local Development

```bash
npm start
```

Opens a local development server at `http://localhost:3000/`.

### Build

```bash
npm run build
```

Generates static content into the `build` directory.

### Linting

```bash
npx markdownlint docs/
npx prettier --check .
```
